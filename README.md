# Seismic-Card

Profile Card Generator for the Seismic community. Generates customizable retro-neon style profile cards as static PNGs and animated GIFs.

## Features
- Dynamic avatar fetching (supports X/Twitter usernames via unavatar/microlink).
- Custom badge styling for Magnitude roles (Magnitude 1.0 - 9.0).
- High-fidelity canvas rendering for PNG and animated GIF outputs.
- Local storage persistence for user card serial numbers.
- Dynamic particle systems and animated ring shimmers.
